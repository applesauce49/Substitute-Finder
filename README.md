# Substitute Finder

Substitute Finder helps teams coordinate temporary coverage quickly, clearly, and with less back-and-forth.

It gives organizations one place to post coverage requests, review applicants, assign work, and track fulfillment activity.

The project originated in the education staffing space, and has since evolved into a broader coverage coordination platform.

## Who This Is For

- Operations and scheduling coordinators
- Team members posting coverage requests
- Staff members applying for open assignments

## What You Can Do

- Post and manage substitute requests
- Browse and apply to available assignments
- Track assigned work and request status
- View calendar-aligned schedules
- Use role-aware dashboards for day-to-day workflows
- Review fulfillment metrics from the admin area

## Product Highlights

- Real-time updates for job lifecycle events
- Google sign-in flow for streamlined access
- Calendar-aware scheduling workflows
- Reporting and metrics for operational visibility
- Admin controls for rules, settings, and match-engine behavior

## Typical Workflow

1. A requester creates a substitute request.
2. Eligible substitutes review and apply.
3. Admin/reviewer assigns coverage and tracks completion.
